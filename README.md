## üìå Project Overview

**WanderLust Adventures** is a modern travel-themed website that showcases:

- Hero landing section
- About section
- Core services section
- Destination showcase grid
- Booking contact form
- Smooth scrolling navigation
- Scroll Spy functionality (Active link highlighting)

The primary goal of this project was to build a visually clean and interactive landing page using only core web technologies without external frameworks.
